# RBTL

RBTL is a Laravel, Vue, Inertia, and Tailwind website for Rudra Beads Testing Lab.

## Pages

- `/` - Home
- `/verify-certificate` - Certificate verification
- `/contact` - Contact

## Development

```bash
composer install
npm install
php artisan serve
npm run dev
```

## Verification

```bash
npm run build
php artisan test
```
